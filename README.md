# Nikki

An agent-ready personal vault starter.

## Start

1. Create your private copy by [clicking this link](https://github.com/new?template_owner=arlegotin&template_name=nikki&visibility=private&name=nikki): make sure GitHub shows **Private**. No GitHub account? [Download the ZIP](https://github.com/arlegotin/nikki/archive/refs/heads/main.zip) and unzip it somewhere private instead.

2. Open the repo folder in Codex, Claude, your terminal agent, or your IDE.
3. Say:

```text
Help me make this my personal vault
```
