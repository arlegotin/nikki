# Journal Instructions

Use this folder for reflection, daily entries, incidents, and decisions.

When writing journal notes:

- Use the current local date from `current-datetime` when date-sensitive.
- Put daily entries in `Journal/Daily/YYYY-MM-DD.md`.
- Put incident or thought records in `Journal/Incidents/YYYY-MM-DD - topic.md`.
- Put decision journals in `Journal/Decisions/YYYY-MM-DD - topic.md`.
- Update `Journal/Index.md` after writing an entry.
- Keep summaries private, concise, and factual.
