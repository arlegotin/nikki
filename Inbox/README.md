# Inbox

Drop unsorted notes here, then periodically move them into the right folder.

Suggested triage:

- Is this an action? Move to a project or create a task.
- Is this reference material? Move to `Resources/`, `Learning/`, or `Media/`.
- Is this personal reflection? Move to `Journal/` or `Me/`.
- Is this stale? Move to `Archive/` or delete it.
